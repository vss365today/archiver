# #vss365 today Archiver
> Generate a static, flat file version of #vss365 today

## Requirements

- Python 3.11+
- Poetry 1.6.1+

### Required Secrets
- API_DOMAIN
- DIST_PATH
- SITE_DOMAIN
- SYS_VARS_PATH (in env)


## Usage
- `poetry install`
- `poetry shell`
- `python ./archiver.py`
